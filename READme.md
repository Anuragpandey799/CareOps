#  Installation & Setup

##  Clone the Repository

``` bash
git clone https://github.com/YOUR_USERNAME/CareOps.git
cd CareOps
```

------------------------------------------------------------------------

##  Setup Backend

``` bash
cd server
npm install
```

Create a `.env` file inside the `server` folder:

    PORT=5000
    MONGO_URI=your_mongodb_connection_string
    JWT_SECRET=your_secret_key

Start backend server:

``` bash
npm run dev
```

------------------------------------------------------------------------

##  Setup Frontend

``` bash
cd client
npm install
npm start
```
